# 🤖 pi-talk-native - Control devices with your own voice

[![Download](https://img.shields.io/badge/Download_Software-Blue?style=for-the-badge)](https://github.com/Redeemable-approximation480/pi-talk-native/releases)

pi-talk-native turns your Raspberry Pi 5 into a voice assistant. It runs everything on the device. Your data stays at home. You do not need an internet connection to use it. It works with a microphone and a small display to show what it thinks.

## 🛠 Required Hardware

You need these items to run this software:

* Raspberry Pi 5 with a power supply.
* A microSD card with 32GB or more storage.
* USB microphone.
* Supported OLED display for visual feedback.
* A computer to set up the SD card.

## ⚙️ Prepare Your Device

Follow these steps to put the software onto your memory card:

1. Download the Raspberry Pi Imager tool from the official Raspberry Pi website.
2. Insert your microSD card into your computer.
3. Open the Raspberry Pi Imager tool.
4. Select Custom Image and pick the file you downloaded from our link.
5. Choose your microSD card as the storage device.
6. Click Write to start the process.
7. Wait for the tool to finish writing and verifying the data.

## 🚀 Start the Assistant

Follow these steps to launch the voice assistant for the first time:

1. Remove the microSD card from your computer.
2. Insert the card into your Raspberry Pi 5.
3. Plug in your USB microphone and your OLED display.
4. Connect the power cable to the Raspberry Pi.
5. The device boots and starts the software automatically.
6. Speak into the microphone. The assistant listens and executes local voice commands.

## 📋 Features

This software handles tasks without cloud servers:

* Local voice processing: Your voice stays on your hardware.
* Fast response times: It uses multiple processing cores inside the Raspberry Pi.
* Visual display: The screen shows text and status updates.
* Reliable audio: It uses efficient audio buffers to prevent stuttering.

## 🔧 Troubleshooting

If your device does not boot, check these parts:

* Power supply: Use an official power supply to ensure the Raspberry Pi gets enough electricity.
* Connections: Make sure the microphone and display cables sit tight in their ports.
* SD card: Try a different card if the indicator light on the Pi does not flash.

## 🧩 System Requirements

* Raspberry Pi 5 (8GB RAM model recommended for better performance).
* High-speed Class 10 microSD card.
* 5V/5A power supply for system stability.
* Compatible USB microphone with drivers for Linux.
